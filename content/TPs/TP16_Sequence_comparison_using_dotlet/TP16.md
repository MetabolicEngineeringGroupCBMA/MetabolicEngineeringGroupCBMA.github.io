**Sequence comparison using dotlet**

Dotlet is a simple to use program that implements the [dotplot algorithm](http://en.wikipedia.org/wiki/Dot_plot_\(bioinformatics\)) with a [substitution matrix](http://en.wikipedia.org/wiki/Substitution_matrix) for protein comparison. This [paper](http://www.ncbi.nlm.nih.gov/pubmed/10842741) describes the Dotlet software.

[https://dotlet.vital-it.ch](https://dotlet.vital-it.ch/) You should see a window like Figure 1
