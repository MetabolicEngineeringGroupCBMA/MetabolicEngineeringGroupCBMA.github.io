Assembly of a pTAx vectors by in-vivo homologous recombination

The pTA7 vector is a circular plasmid assembled from five different linear PCR products. Table 2 shows the five PCR products with templates and primer numbers. The first column in Table 2 contains the designation for each of the five PCR products. The Template column contain the designation of the template sequence. Sequence files corresponding to this column (with “.gb” file endings can be found in the same folder as this file. The Primer#1 and #2 columns contain two numbers corresponding to the first column in Table 4, where you can find the primer sequences.
