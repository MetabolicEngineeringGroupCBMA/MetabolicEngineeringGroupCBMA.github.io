Sometimes the rendering of Jupyter notebooks on GitHub does not work and you get something like this:

![](norender.png)

One possible workaround is to download the notebook to your computer and open it using
the [IPython/Jupyter Notebook Previewer](https://jsvine.github.io/nbpreview).

You can even drag the file from you computer onto the web page with no typing at all.
This is the result from the example above.

![](render.png)
